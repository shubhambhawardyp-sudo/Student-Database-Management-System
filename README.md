# Student Database Management System

A simple Student Database Management System developed using Python and SQLite.

## Features

- Add Student
- View Student Records
- Search Student
- Update Student Information
- Delete Student Records

## Technologies Used

- Python
- SQLite

## Database Operations

This project implements CRUD operations:

- Create (Add Student)
- Read (View/Search Student)
- Update (Modify Student Details)
- Delete (Remove Student)

## How to Run

1. Install Python
2. Download the project
3. Run:

python main.py

## Learning Outcomes

- Database Management Concepts
- SQL Queries
- CRUD Operations
- Python Database Connectivity
